# Calendar.app

NeolyxOS Calendar Application with themes, clock faces, and live icon.

## Features

- **7 Themes**: Midnight, Arctic, Sunset, Forest, Ocean, Sakura, Monochrome
- **6 Clock Faces**: Digital, Analog, Minimal, Roman, Binary, Flip
- **Live Icon**: Shows current date in app icon
- **Navigation Bar**: Click clock to launch Calendar

## Themes

| Theme | Description |
|-------|-------------|
| Midnight | Dark blue/purple (default) |
| Arctic | Light icy blue |
| Sunset | Warm orange/red |
| Forest | Natural green |
| Ocean | Teal/aqua |
| Sakura | Cherry blossom pink |
| Monochrome | Black and white |

## Clock Faces

1. **Digital** - Large digital display (HH:MM:SS)
2. **Analog** - Traditional analog with numbers
3. **Minimal** - Analog without numbers
4. **Roman** - Roman numeral markers
5. **Binary** - Binary clock (6 columns)
6. **Flip** - Flip clock style panels

## Build

```bash
cd apps/Calendar.app
make
```
