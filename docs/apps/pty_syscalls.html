PTY Syscalls

NeolyxOS Pseudo-Terminal Subsystem

Overview

The PTY subsystem provides terminal multiplexing for graphical terminal emulators like Terminal.app. It creates master/slave fd pairs with ring buffers for bidirectional I/O.

Syscall Numbers

Number Syscall Signature
60 SYS_PTY_OPEN int pty_open(int* master, int* slave)
61 SYS_PTY_READ ssize_t pty_read(int fd, void* buf, size_t count)

63 SYS_PTY_CLOSE int pty_close(int fd)
64 SYS_PTY_RESIZE int pty_resize(int fd, int cols, int rows)
65 SYS_PTY_IOCTL int pty_ioctl(int fd, int cmd, void* arg)

67 SYS_PTY_POLL int pty_poll(int fd, int timeout_ms)

Data Flow

Terminal.app (Userspace)
      │
      │ pty_write(master_fd, input)
      ↓
┌─────────────────────┐
│   Master Buffer     │ → Ring buffer (4KB)
└─────────────────────┘
      ↓
┌─────────────────────┐
│   Shell Process     │ (reads from slave_fd)
│    (/bin/nxsh)      │
└─────────────────────┘
      ↓
┌─────────────────────┐
│   Slave Buffer      │ → Ring buffer (4KB)
└─────────────────────┘
      ↓
      │ pty_read(master_fd, output)
Terminal.app (Userspace)

IOCTL Commands

Command Value Description
PTY_IOCTL_GET_SIZE 1 Get terminal size (rows, cols)
PTY_IOCTL_SET_SIZE 2 Set terminal size
PTY_IOCTL_GET_OWNER 3 Get owning process PID
PTY_IOCTL_SET_ECHO 4 Enable/disable echo
PTY_IOCTL_SET_RAW 5 Set raw mode
PTY_IOCTL_FLUSH 6 Flush buffers

Flags

Flag Value Description
PTY_FLAG_ECHO 0x01 Echo input back
PTY_FLAG_RAW 0x02 Raw mode (no line buffering)
PTY_FLAG_CRLF 0x04 Convert \n to \r\n
PTY_FLAG_CTRLC 0x08 Handle Ctrl+C as SIGINT

Example Usage

#include "io/pty.h"

int master_fd, slave_fd;

// Create PTY pair
if (pty_open(&master_fd, &slave_fd) < 0) {
    return -1;
}

// Spawn shell
int shell_pid = pty_spawn_shell(slave_fd, "/bin/nxsh");

// Set terminal size
pty_resize(master_fd, 80, 24);

// Write to shell
const char *cmd = "ls -la\n";
pty_write(master_fd, cmd, strlen(cmd));

// Read shell output
char buffer[4096];
ssize_t n = pty_read(master_fd, buffer, sizeof(buffer));

// Cleanup
pty_close(master_fd);
pty_close(slave_fd);
