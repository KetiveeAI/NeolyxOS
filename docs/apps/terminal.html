Features

🗂️ Multiple Tabs

Cmd+T new tab, Cmd+W close.

🔌 Full PTY

Real pseudo-terminal with syscalls 60-67.

🎨 ANSI Colors

256 color support with custom themes.

📜 Scrollback

10,000 lines of history.

✂️ Copy/Paste

Selection and clipboard support.

🎨 Themes

Dark and Light themes included.

File Purpose

PTY Syscalls

Syscall # Purpose
SYS_PTY_OPEN 60 Create PTY pair
SYS_PTY_READ 61 Read from master
SYS_PTY_WRITE 62 Write to master
SYS_PTY_CLOSE 63 Close PTY
SYS_PTY_RESIZE 64 Update window size
SYS_PTY_IOCTL 65 Terminal control
SYS_PTY_SPAWN 66 Spawn shell
SYS_PTY_POLL 67 Check for input

Building

cd apps/Terminal.app

# Standalone build
make

# With NXRender graphics
make real

# Install to /Applications
make install
