Path.app Complete

NeolyxOS File Manager — A Finder-like file browser with tabs, Quick Look, and themes.

Directory Structure

/apps/Path.app/
├── bin/
│   └── path                 # Main binary
├── lib/
│   └── libpath.so          # File operations library
├── resources/
│   ├── icons/              # File type icons (NXI format)
│   │   ├── folder.nxi
│   │   ├── file.nxi
│   │   └── image.nxi
│   ├── themes/

Features

🗂️ Multiple Tabs

Ctrl+T new tab, Ctrl+W close.

👁️ Quick Look

Press Space to preview files.

ℹ️ Get Info

Press Cmd+I for file details.

🎨 Themes

Dark and Light themes with NXI icons.

📊 4 View Modes

Icons, List, Columns, Gallery.

⭐ Sidebar

Favorites, Locations, Volumes.

Keyboard Shortcuts

Category Shortcut Action
Tabs Ctrl+T New tab
Ctrl+W Close tab
Ctrl+Tab Next tab
Files Cmd+C Copy
Cmd+X Cut
Cmd+V Paste
Delete Move to Trash
View Space Quick Look
Cmd+I Get Info
Cmd+. Toggle hidden

Source Files

File Purpose
main.c App lifecycle, navigation
file_view.c Icons, List, Columns, Gallery
sidebar.c Favorites, Volumes panel
toolbar.c Navigation, view controls
context_menu.c Right-click menu
tab_bar.c Multi-tab management
quick_look.c Space key preview
get_info.c File info panel

Manifest (manifest.npa)

{
    "name": "Path",
    "bundle_id": "com.neolyx.path",
    "binary": "bin/path",
    "library": "lib/libpath.so",
    "icon": "resources/path.nxi",
    "permissions": [
        "filesystem.read",
        "filesystem.write",
        "storage.volumes"
    ]
}

Building

cd apps/Path.app

# Standalone build
make

# With NXRender graphics
make real

# Install to /Applications
make install
