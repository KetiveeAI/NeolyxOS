Application Directory Structure

/apps/<AppName>.app/
├── bin/
│   └── <binary>            # Main executable
├── lib/
│   └── lib<name>.so        # Shared library
├── resources/
│   ├── icons/              # File type icons (NXI format)
│   ├── themes/             # Theme files (.theme)
│   └── <app>.nxi           # App icon
├── manifest.npa            # App manifest (JSON)
├── Makefile
└── README.md

Manifest Format (manifest.npa)

{
    "name": "AppName",
    "version": "1.0.0",
    "bundle_id": "com.neolyx.appname",
    "category": "Utilities",
    "description": "App description",
    
    "binary": "bin/appname",
    "library": "lib/libappname.so",
    "icon": "resources/appname.nxi",
    
    "permissions": [
        "filesystem.read",
        "filesystem.write"
    ],
    
    "menu": {
        "file": [...],
        "edit": [...],
        "view": [...]
    }
}

Icon Format (NXI)

# NeolyxOS NXI Vector Icon Format

[icon]
name = "appname"
size = 64

[colors]
primary = #89B4FA
secondary = #CDD6F4

[paths]
shape = "M0,0 L64,0 L64,64 L0,64 Z"
shape_fill = primary

Theme Format (.theme)

# NeolyxOS Theme Format

[theme]
name = "dark"
variant = "dark"

[window]
background = #1E1E2E
titlebar = #181825

[text]
primary = #CDD6F4
dim = #6C7086
