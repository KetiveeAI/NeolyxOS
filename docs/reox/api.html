REOX Standard Library

Built-in functions and system APIs for REOX applications.

Core Functions

Function Description Example
print(args...) Print to console print("Hello", x)
len(collection) → int Get length len([1,2,3]) // 3
type(value) → string Get type name type(42) // "int"
to_string(value) → string Convert to string to_string(42) // "42"
parse_int(str) → int Parse integer parse_int("42") // 42
parse_float(str) → float Parse float parse_float("3.14")

Math Functions

Import: import std.math

Function Description
abs(x) → int/float Absolute value
min(a, b) → int/float Minimum value
max(a, b) → int/float Maximum value
pow(x, n) → float Power (x^n)
sqrt(x) → float Square root
sin(x) → float Sine
cos(x) → float Cosine
floor(x) → int Round down
ceil(x) → int Round up
random() → float Random 0.0-1.0

String Functions

Import: import std.string

Function Description
split(s, delim) → [string] Split string
join(arr, sep) → string Join array
trim(s) → string Trim whitespace
upper(s) → string Uppercase
lower(s) → string Lowercase
contains(s, sub) → bool Contains substring
starts_with(s, prefix) → bool Starts with
ends_with(s, suffix) → bool Ends with
replace(s, old, new) → string Replace all

Array Methods

Method Description
arr.push(item) Add to end
arr.pop() → item Remove from end
arr.first() → maybe item Get first
arr.last() → maybe item Get last
arr.map(fn) → [T] Transform each
arr.filter(fn) → [T] Filter by predicate
arr.find(fn) → maybe T Find first match
arr.sort() → [T] Sort ascending
arr.reverse() → [T] Reverse order

System APIs (NeolyxOS)

Filesystem

Import: import system.fs

Function Description
fs.read(path) → string Read file contents
fs.write(path, data) Write to file
fs.exists(path) → bool Check if exists
fs.list_dir(path) → [Entry] List directory
fs.mkdir(path) Create directory
fs.remove(path) Delete file/dir

Process

Import: import system.process

Function Description
process.spawn(cmd) → Process Start process
process.exec(cmd) → string Run and get output
process.exit(code) Exit program
process.args() → [string] Get CLI arguments

Monitor

Import: import system.monitor

Function Description
monitor.get_cpu() → float CPU usage %
monitor.get_memory() → float Memory usage %
monitor.get_disk() → float Disk usage %
monitor.get_uptime() → int Uptime in seconds

C FFI (Foreign Function Interface)

Call C functions directly from REOX.

extern "C" {
    fn nx_gpu_fill_rect(x: int, y: int, w: int, h: int, color: int);
    fn nx_audio_play(buffer: [int], len: int);
    fn nx_net_connect(host: string, port: int) -> int;
}
