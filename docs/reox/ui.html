REOX UI System

REOX provides a declarative UI framework for building native NeolyxOS applications.

Core Concepts

Term Description
layer A UI component (like a widget or view)
panel A top-level window container
@Bind Reactive state binding
emit Dispatch an event
signal Observable reactive value

Creating a Layer

Layers are the building blocks of REOX UIs.

layer Greeting {
    text("Hello, NeolyxOS!")
        .color(0x58A6FF)
        .font_size(24)
        .padding(16)
}

Creating a Panel (Window)

panel MyApp {
    title: "My Application"
    size: (800, 600)
    
    vstack(gap: 16) {
        text("Welcome!")
        button("Click Me")
    }
}

State with @Bind

Use @Bind for reactive state that updates the UI automatically.

panel Counter {
    title: "Counter"
    
    @Bind count: int = 0
    
    vstack(gap: 20) {
        text("Count: " + count)
            .font_size(32)
        
        hstack(gap: 10) {
            button("-").on_click(action { count = count - 1 })
            button("+").on_click(action { count = count + 1 })
        }
    }
}

Built-in Widgets

Widget Usage Description
text(str) text("Hello") Display text
button(str) button("Click") Clickable button
input() input().placeholder("...") Text input field
checkbox(bool) checkbox(true) Toggle checkbox
slider(min,max) slider(0, 100) Value slider
image(path) image("/icons/app.png") Display image
divider() divider() Horizontal line

Layout Containers

Container Usage Description
vstack vstack(gap: 10) { ... } Vertical stack
hstack hstack(gap: 10) { ... } Horizontal stack
zstack zstack() { ... } Layered/overlapping
scroll scroll() { ... } Scrollable container
grid grid(cols: 3) { ... } Grid layout

Modifiers

Chain modifiers to style widgets.

text("Styled Text")
    .color(0xFFFFFF)
    .background(0x1E1E1E)
    .font_size(18)
    .font_weight("bold")
    .padding(16)
    .corner_radius(8)
    .shadow(4)
    .opacity(0.9)
    .on_click(action { print("Clicked!") })
    .on_hover(action { print("Hovered!") })

Events

button("Submit")
    .on_click(action {
        print("Button clicked!");
        emit submit_form();
    })
    .on_long_press(action {
        print("Long pressed!");
    })

Conditional Rendering

layer StatusBadge {
    @Bind is_online: bool = false
    
    if is_online {
        text("Online").color(0x3FB950)
    } else {
        text("Offline").color(0xF85149)
    }
}

Lists

layer ItemList {
    @Bind items: [string] = ["Apple", "Banana", "Cherry"]
    
    scroll() {
        for item in items {
            text(item)
                .padding(12)
                .background(0x21262D)
                .corner_radius(6)
        }
    }
}
