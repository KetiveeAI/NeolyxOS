What is REOX?

REOX is a modern programming language designed specifically for NeolyxOS. It provides:

  • Simple, safe syntax (simpler than C++, safer than C)
  • Native UI framework for system and application design
  • Zero-cost abstractions for OS-level performance
  • Declarative UI with reactive state bindings

Quick Start

fn main() {
    print("Hello from REOX!");
}

Build & Run

# Compile REOX source
reoxc build app.rx -o app

# Run with interpreter
reoxc run app.rx

# Start REPL
reoxc repl
