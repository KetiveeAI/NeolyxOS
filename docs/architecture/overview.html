📐 NeolyxOS System Architecture

A custom 64-bit operating system built from scratch.

System Stack

┌─────────────────────────────────────────────────────────────┐
│                     USER SPACE                              │
│  Terminal │ ReoLab IDE │ Custom Apps │ REOX UI Apps         │
├─────────────────────────────────────────────────────────────┤
│                     GUI LAYER                               │
│  NXRender + NXGFX Engine                                    │
│  • Custom GPU Backend  • Widget System  • Animation         │
├─────────────────────────────────────────────────────────────┤
│                     KERNEL LAYER                            │
│  Core      │ Drivers   │ File System │ Memory Mgmt          │
│  Syscalls  │ ATA/NVMe  │ VFS/NXFS    │ Heap/Paging          │
│  Process   │ Network   │ FAT32       │ PMM                  │
├─────────────────────────────────────────────────────────────┤
│                     BOOT LAYER                              │
│  UEFI Bootloader → ELF Loading → Boot Info → kernel_main()  │
├─────────────────────────────────────────────────────────────┤
│                     HARDWARE                                │
│  CPU (x86_64) │ RAM │ GPU │ Storage │ Network │ Input       │
└─────────────────────────────────────────────────────────────┘

Directory Structure

neolyx-os/
├── boot/              # UEFI Bootloader
│   ├── main.c         # Kernel loading, graphics
│   └── graphics.c     # GOP splash screen
│
├── kernel/            # Kernel source
│   ├── kernel_main.c  # Entry point
│   └── src/
│       ├── arch/x86_64/  # GDT, IDT, paging
│       ├── core/         # Process, syscall
│       ├── drivers/      # Storage, network, input
│       ├── fs/           # VFS, NXFS
│       └── mm/           # Heap, PMM
│
├── gui/               # GUI Layer
│   └── nxrender/      # Custom rendering engine
│
├── REOX/              # REOX Language
│   └── reolab/        # Compiler/Runtime
│
└── docs/              # Documentation (this)

Boot Info Structure

typedef struct {
    uint32_t magic;               // 0x4E454F58 "NEOX"
    uint32_t version;
    uint64_t memory_map_addr;
    uint64_t memory_map_size;
    uint64_t framebuffer_addr;
    uint32_t framebuffer_width;
    uint32_t framebuffer_height;
    uint32_t framebuffer_pitch;
    uint64_t acpi_rsdp;
} NeolyxBootInfo;

Build Commands

# Full build
./build.sh

# Kernel only
cd kernel && make

# Test in QEMU
./test_boot.sh

System Calls

# Name Description
0 sys_exit Terminate process
1 sys_read Read from file descriptor
2 sys_write Write to file descriptor
3 sys_open Open file
4 sys_close Close file
5 sys_mmap Map memory
6 sys_fork Create child process
7 sys_exec Execute program
