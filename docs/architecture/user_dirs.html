👤 User Directory Structure

Personal files organized per-user under /Users/{username}/

Home Directory Layout

/Users/{username}/ │ ├── 🖥️Desktop/ — Desktop files and folders ├── 📄Documents/ — Personal documents ├── ⬇️Downloads/ — Downloaded files ├── 🖼️Pictures/ — Photos and images ├── 🎵Music/ — Audio files ├── 🎬Videos/ — Video files ├── 🌐Public/ — Shared with network (read-only) ├── 📦Applications/ — User-installed apps │ ├── 📚Library/ — User preferences and app data │ ├── Preferences/ — App settings (.plist) │ ├── Caches/ — App caches │ └── Logs/ — User app logs │ └── .config/ — Hidden config files ├── shell/ — Terminal settings └── nxrender/ — Theme settings

Standard Directories

Directory Path Variable Purpose
Desktop $HOME/Desktop Desktop icons and files
Documents $HOME/Documents User documents
Downloads $HOME/Downloads Browser/app downloads
Pictures $HOME/Pictures Photos and screenshots
Music $HOME/Music Audio files
Videos $HOME/Videos Video files
Public $HOME/Public Network-shared folder

Username in Terminal

# Terminal prompt format
{username}@{hostname}:{current_dir}$

# Example
alice@neolyx:~/Documents$

# Get current user
whoami
→ alice

# Get home directory
echo $HOME
→ /Users/alice

Public Folder (Network Sharing)

Files in /Users/{username}/Public/ are visible to other devices on the network.

  • Read-only by default for network users
  • Drop Box/ subfolder allows write access
  • Enable sharing in Settings → Sharing
# Enable file sharing
nxconfig sharing enable

# Set network name
nxconfig set hostname "Alice-MacBook"

# Other devices see:
# nxshare://Alice-MacBook/alice/Public

User Creation

# Create new user
nxuser add bob --fullname "Bob Smith"

# Creates:
# /Users/bob/
# /Users/bob/Desktop/
# /Users/bob/Documents/
# ... all standard directories

# Set password
nxuser passwd bob

# Make admin
nxuser mod bob --admin
