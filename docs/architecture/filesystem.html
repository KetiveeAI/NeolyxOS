Protection Levels

Path Protection Who Can Modify
/System/ SIP Recovery Mode only
/Kernel/ SIP Recovery Mode only
/Boot/ SIP Recovery Mode only
/Applications/ Admin Admin users
/Library/ Admin Admin users
/Users/{user}/ User Owner only
/tmp/ Open All processes

Special Directories

/System/Library/

Core OS frameworks and services. Never modify directly.

  • CoreServices/ — Window server, login
  • Frameworks/ — NXRender, NXAudio, etc.
  • Extensions/ — Kernel extensions

/Volumes/

All mounted filesystems appear here.

  • Boot drive: /Volumes/NeolyxOS
  • USB drives auto-mount here
  • Network shares mount here

Device Name & Hostname

# Terminal prompt uses username
user@neolyx:~$ 

# Set device name (used for network)
nxconfig set hostname "MyMachine"

# Get current hostname
nxconfig get hostname
→ MyMachine
