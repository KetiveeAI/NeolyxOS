Critical Fixes

GPF SYSRET/GDT Fix Jan 2026

Fixed triple fault caused by SYSRET loading TSS as code segment due to incorrect GDT layout. User CS selector 0x2B was pointing to TSS descriptor instead of User Code 64 segment.

Documentation Standards

Each bug fix document includes:

  • Symptom - What the user/developer observed
  • Root Cause - Technical explanation of why it happened
  • Before/After - What changed (GDT layout, code, etc.)
  • Files Modified - Which source files were touched
  • Verification - How to confirm the fix works
  • Key Lesson - Takeaway for preventing similar bugs
