🚫 What NOT to Do (Stay Focused!)

❌ Don't Touch Why
AMD Zen optimizations Stubs are fine, optimize later
Bluetooth driver Not critical for boot
Network stack completion TCP/IP works, sockets can wait
NTFS/APFS improvements FAT32 + NXFS is enough
3D GPU acceleration Framebuffer is sufficient
Recovery mode polish Works as-is

Rule: If it's not on the critical path → SKIP IT
