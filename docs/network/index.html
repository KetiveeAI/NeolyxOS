🌐 NeolyxOS Network Subsystem

Status: Kernel-level TCP/IP Stack & Drivers (Partially Userspace compatible)

TCP/IP Stack Architecture

Details on the custom kernel network stack, including Sockets, DHCP, Firewall, and Protocols (TCP/UDP/ICMP/IPv4).

Network Drivers

Supported hardware including Intel E1000, VirtIO, Realtek NICs, and WiFi chipsets.

Overview

The NeolyxOS network stack is a custom implementation written from scratch in C. It resides primarily in the kernel but exposes a standard BSD-like Socket API to userspace.

Key Features

  • IPv4 Only: Full IPv4 support (ARP, ICMP, Routing).
  • Stateful TCP: Connection-oriented reliable transport.
  • Built-in DHCP: Automatic IP configuration.
  • Integrated Firewall: Packet filtering at the IP layer.
