🔌 Network Hardware Support

NeolyxOS uses a unified kernel driver (nxnet.kdrv) that abstracts various hardware NICs into a common interface.

Architecture

The system uses detect_network_devices() to scan the PCI bus for Class 0x02 (Network). Recognized cards are automatically initialized and registered as eth0 through ethN.

Supported Ethernet Controllers (Wired)

Intel E1000

82540EM (QEMU)

Status: Stable

Full MMIO driver with TX/RX ring buffers.

Intel E1000e

82574L (PCIe)

Status: Stable

PCIe variant of E1000.

VirtIO-Net

1AF4:1000

Status: Stable

Paravirtualized driver for high-performance VM networking.

Realtek

RTL8139 / RTL8169

Status: Beta

Legacy 10/100 support.

📡 WiFi Support

Source: kernel/src/drivers/network/wifi.c

WiFi support is integrated via a generic Firmware Abstraction Layer. This allows supporting multiple vendors with a consistent API.

Supported Chipsets

  • Intel: 8260 series
  • Broadcom: BCM4360 series
  • Atheros: AR9280 series
  • Realtek: RTL8821CE series
