NXDisplay Advanced API

DDC/CI monitor control, Adaptive Refresh, Gaming Mode, Night Light

Features Overview

Feature Purpose
DDC/CI Control external monitor OSD (brightness, contrast)
Night Light Blue light filter (software gamma)
Adaptive Refresh Power-saving dynamic Hz
Gaming Mode VRR + HDR + low latency

DDC/CI Monitor Control

Control external monitor hardware settings via I2C:

#include "drivers/nxdisplay_kdrv.h"

// Check if monitor supports DDC/CI
if (nxdisplay_kdrv_ddc_supported(0)) {
    // Get hardware brightness
    int brightness = nxdisplay_kdrv_ddc_get_brightness(0);
    
    // Set brightness (0-100)
    nxdisplay_kdrv_ddc_set_brightness(0, 70);
    
    // Set contrast
    nxdisplay_kdrv_ddc_set_contrast(0, 50);
    
    // Switch input source
    nxdisplay_kdrv_ddc_set_input(0, NXDDC_INPUT_HDMI1);
}

DDC/CI Commands

Function VCP Code
Brightness 0x10
Contrast 0x12
Color Temperature 0x14
Input Source 0x60

Night Light (Blue Light Filter)

// Enable with color temperature (Kelvin)
nxdisplay_kdrv_night_enable(0, 3400);  // Warm (evening)
nxdisplay_kdrv_night_enable(0, 4500);  // Moderate
nxdisplay_kdrv_night_enable(0, 6500);  // Off (daylight)

// Disable
nxdisplay_kdrv_night_disable(0);

// Check status
if (nxdisplay_kdrv_night_active(0)) {
    int kelvin = nxdisplay_kdrv_night_get_temperature(0);
}

Adaptive Refresh Rate

Dynamic Hz based on workload (laptop power saving):

// Set mode
nxdisplay_kdrv_adaptive_set_mode(0, NXADAPT_BALANCED);

// Modes:
// NXADAPT_OFF         - Fixed Hz
// NXADAPT_POWER_SAVE  - Aggressive (30-60Hz idle)
// NXADAPT_BALANCED    - Moderate (60Hz default)
// NXADAPT_PERFORMANCE - High Hz priority

Workload → Refresh Rate

Workload Target Hz
Idle 30 Hz
Browsing 60 Hz
Video 60 Hz (match content)
Productivity 90 Hz
Gaming Max (144/165/240)

Gaming Mode

One-click optimization for games:

nxgame_config_t config = {
    .mode = NXGAME_MODE_ON,
    .vrr_enabled = 1,        // Variable Refresh Rate
    .hdr_enabled = 1,        // HDR
    .low_latency = 1,        // Reduce input lag
    .disable_compositor = 1, // Bypass compositing
    .target_hz = 0,          // Max available
    .vrr_min_hz = 48         // VRR minimum
};

nxdisplay_kdrv_game_mode_enable(0, &config);

// Register games for auto-detection
nxdisplay_kdrv_game_register("steam.exe");
nxdisplay_kdrv_game_register("game.app");
