NXGame ABI v1.0 FROZEN

Permanently frozen. Engines can rely on backward compatibility.

⚠️ ABI Stability Contract

Syscall numbers, struct layouts, and capability flags are permanently frozen. Breaking changes require new major version.

Frozen Syscall Numbers (110–119)

Number Name Description
110 SYS_NXGAME_SESSION_CREATE Create exclusive session
111 SYS_NXGAME_SESSION_DESTROY Destroy session
112 SYS_NXGAME_SESSION_START Start session
113 SYS_NXGAME_SESSION_STOP Stop session
114 SYS_NXGAME_GPU_ACQUIRE Request exclusive GPU
115 SYS_NXGAME_GPU_RELEASE Release GPU
116 SYS_NXGAME_GPU_PRESENT Flip buffers
117 SYS_NXGAME_AUDIO_SUBMIT Submit audio
118 SYS_NXGAME_INPUT_POLL Poll input
119 SYS_NXGAME_QUERY_CAPS Query capabilities

Platform Capability Flags

Flag Value Meaning
NXGAME_PLAT_CAP_VSYNC 0x0001 VSync supported
NXGAME_PLAT_CAP_EXCLUSIVE 0x0002 Exclusive mode
NXGAME_PLAT_CAP_AUDIO_LL 0x0004 Low-latency audio
NXGAME_PLAT_CAP_INPUT_RAW 0x0008 Raw input polling
NXGAME_PLAT_CAP_GAMEPAD 0x0010 Gamepad support
NXGAME_PLAT_CAP_DOUBLE_BUFFER 0x0020 Double buffering

Usage Example

nxgame_caps_t caps;
nxgame_query_caps(&caps);

if (caps.platform_caps & NXGAME_PLAT_CAP_EXCLUSIVE) {
    // Use exclusive mode
}

nxgame_session_t* session = nxgame_session_create(
    NXGAME_SESSION_GAME,
    NXGAME_CAP_GPU | NXGAME_CAP_AUDIO | NXGAME_CAP_INPUT,
    NULL
);
nxgame_session_start(session);
// ... game loop ...
nxgame_session_destroy(session);
