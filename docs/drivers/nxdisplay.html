Capability Descriptor

Query once, cache in NXRender/NXGame:

nxdisplay_caps_t caps;
nxdisplay_kdrv_get_caps(0, &caps);

// Supported features
caps.supports.vrr       // VRR capable
caps.supports.hdr10     // HDR10 capable
caps.supports.audio     // Audio over display

// Currently active
caps.active.vrr         // VRR enabled
caps.active.hdr         // HDR enabled

// Refresh range
caps.min_refresh_hz     // 24 Hz
caps.max_refresh_hz     // 144 Hz

Declarative Mode Request

Apps request features, OS arbitrates:

// NXGame says: "I want VRR + HDR if possible"
nxdisplay_mode_hint_t hint = {
    .want_vrr = 1,
    .want_hdr = 1,
    .want_low_latency = 1,
    .target_refresh_hz = 120
};

nxdisplay_mode_result_t result;
nxdisplay_kdrv_request_mode(0, &hint, &result);

// Check what OS granted
if (result.vrr_enabled) { /* VRR active */ }
if (result.hdr_enabled) { /* HDR active */ }
printf("Refresh: %d Hz", result.actual_refresh_hz);
