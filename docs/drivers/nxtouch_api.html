NXTouch Driver API

Multi-touch gesture recognition for NeolyxOS
Supports: Mobile, Tablet, Laptop (trackpad), Desktop (touchscreen)

Quick Start

#include "drivers/nxtouch.h"

// Initialize
nxtouch_init();

// Check device capabilities
if (nxtouch_is_touchscreen()) {
    printf("Touch screen available\n");
}
if (nxtouch_stylus_available()) {
    printf("Stylus support detected\n");
    nxtouch_palm_rejection_enable(true);  // Auto-enable with stylus
}

// Register gesture callback
nxtouch_register_gesture_handler(on_gesture, NULL);

Gesture Types

Single Finger

Enum Description
NXGESTURE_TAP Single tap (click)
NXGESTURE_DOUBLE_TAP Double tap (zoom)
NXGESTURE_TRIPLE_TAP Triple tap (select paragraph)
NXGESTURE_LONG_PRESS Hold (context menu)
NXGESTURE_SWIPE_* Directional swipe (UP/DOWN/LEFT/RIGHT)
NXGESTURE_FLICK_* Quick velocity-based flick
NXGESTURE_DRAG Drag (touch + move)

Two Finger

Enum Description
NXGESTURE_SCROLL Vertical scroll
NXGESTURE_SCROLL_HORIZONTAL Horizontal scroll
NXGESTURE_PINCH Pinch zoom
NXGESTURE_SPREAD Spread apart (zoom in)
NXGESTURE_ROTATE Two-finger rotate
NXGESTURE_TWO_TAP Right-click

Three/Four Finger

Gesture Default Action
3-finger swipe up/down Volume control
3-finger swipe left/right Brightness control
4-finger swipe up Mission Control
4-finger swipe left/right Switch apps
4-finger tap Show desktop

Edge Gestures (Touch Screen)

Edge Default Action
Left edge Back navigation
Right edge Control Center
Top edge Notifications
Bottom edge App Switcher

Stylus Support

Features

  • 4096 pressure levels - Fine drawing control
  • Tilt X/Y - -90° to 90° detection
  • Barrel rotation - 0-360° for brush angle
  • 2 buttons + eraser - Full tool switching
  • Hover detection - Pre-touch tracking
  • Battery status - For wireless stylus
nxstylus_state_t stylus;
nxtouch_get_stylus(&stylus);

if (stylus.touching) {
    draw_at(stylus.x, stylus.y, stylus.pressure);
    if (stylus.eraser_end) {
        erase_at(stylus.x, stylus.y);
    }
}

Palm Rejection

Automatically enabled when stylus is detected:

nxpalm_config_t palm = {
    .enabled = true,
    .stylus_priority = true,     // Reject touch when stylus in range
    .size_detection = true,      // Detect palm by size
    .min_palm_size = 100,        // Pixels
    .sensitivity = 7             // 1-10
};
nxtouch_palm_set_config(&palm);

App Switcher (Android-style)

nxtouch_appswitcher_t switcher = {
    .enabled = true,
    .edge_width = 40,                    // Pixels from bottom
    .swipe_threshold = 30,               // Min distance
    .swipe_up_close_app = true,          // Swipe up = close
    .swipe_left_right_switch = true,     // L/R = switch apps
    .hold_for_floating = true            // Hold = float window
};
nxtouch_appswitcher_enable(&switcher);
