NXBacklight Driver API

Laptop display brightness and power management
Controls: PWM/ACPI backlight, ambient sensor, keyboard backlight

Backlight Methods

Method Use Case
ACPI Most laptops (acpi_video0)
PWM Direct hardware control
Intel Intel graphics backlight
AMD AMD graphics backlight
Platform Dell, HP, Lenovo specific

Quick Start

#include "drivers/nxbacklight.h"

// Initialize
nxbl_init();

// Check if laptop
if (nxbl_is_laptop()) {
    // Get current brightness
    int brightness = nxbl_get_brightness();  // 0-100%
    
    // Set brightness (smooth transition)
    nxbl_set_brightness_smooth(70, 300);  // 70%, 300ms
    
    // Enable auto-brightness (if ambient sensor available)
    if (nxbl_ambient_available()) {
        nxbl_auto_brightness_enable(true);
    }
}

Brightness Control

Function Description
nxbl_get_brightness() Get brightness 0-100%
nxbl_set_brightness(percent) Set brightness
nxbl_set_brightness_smooth(percent, ms) Smooth transition
nxbl_get_max_brightness() Get raw max level

Auto-Brightness

// Check if ambient light sensor available
if (nxbl_ambient_available()) {
    // Get sensor reading
    nxbl_ambient_t ambient;
    nxbl_get_ambient(&ambient);
    printf("Light: %d lux, suggested: %d%%\n", 
           ambient.lux, ambient.target_brightness);

    // Enable auto-brightness
    nxbl_auto_brightness_enable(true);
}

Keyboard Backlight

if (nxkb_backlight_available()) {
    // Set brightness
    nxkb_set_brightness(50);  // 50%
    
    // Set color (RGB keyboards)
    nxkb_set_color(0xFF00FF);  // Purple
    
    // Auto-off timeout
    nxkb_set_timeout(30);  // 30 seconds
}

// Check capabilities
nxbl_has_lid();             // Laptop lid sensor
nxbl_has_ambient_sensor();  // Light sensor
nxbl_has_accelerometer();   // Orientation sensor
nxbl_has_battery();         // Portable device

Power Management

// Battery notifications
nxbl_on_battery(true);   // Switch to battery
nxbl_on_battery(false);  // On AC power

// Lid events
nxbl_on_lid_close();  // Turn off backlight
nxbl_on_lid_open();   // Restore backlight

// Power profiles
nxbl_set_profile(NXBL_PROFILE_POWER_SAVE);  // Reduce on battery
nxbl_set_profile(NXBL_PROFILE_MOVIE);       // Disable auto-dim

Display Integration

NXBacklight integrates with nxdisplay_kdrv for unified control:

  • Internal display - Uses PWM/ACPI (nxbl_*)
  • External monitor - Uses DDC/CI (nxdisplay_kdrv_ddc_*)
// Unified brightness (auto-detects internal vs external)
int index = nxbl_get_display_index();
nxbl_unified_set_brightness(index, 80);
