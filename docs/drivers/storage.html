Storage Drivers

NeolyxOS includes production-grade storage drivers for AHCI/SATA, NVMe, and ATA devices.

AHCI/SATA Driver

Status: ✅ Production-ready

Features:

  • PCI Discovery & Bus Mastering
  • Port Management (Up to 32 ports)
  • LBA48 Support (Up to 128 PB)
  • Error Handling (Timeout, TFD)

NVMe Driver

Status: ✅ Production-ready with advanced features

  • Multi-queue support (Admin + I/O queues)
  • PRP List handling for scatter-gather DMA
  • MSI-X Interrupt detection

Performance
