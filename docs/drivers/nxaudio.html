Ring Buffer Design

typedef struct {
    uint8_t  data[64KB];      // Audio samples
    uint32_t write_pos;       // Server writes
    uint32_t read_pos;        // Kernel reads  
    uint32_t period_count;    // IRQ counter
    uint8_t  running;
    uint8_t  underrun;
} nxaudio_ring_t;

Server IPC Messages

Type Direction Description
NXMSG_CONNECT Client→Server Connect client
NXMSG_SOURCE_CREATE Client→Server Create audio source
NXMSG_SOURCE_PLAY Client→Server Start playback
NXMSG_LISTENER_SET Client→Server Set 3D listener
NXMSG_SOURCE_ID Server→Client Return new source ID

Spatial Audio Features

  • HRTF - Head-related transfer function for binaural audio
  • Distance Attenuation - Inverse, linear, exponential models
  • Doppler Effect - Pitch shift based on velocity
  • Panning - Constant-power stereo panning fallback
