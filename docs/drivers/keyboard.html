Special Keys

ALLOW Key

Physical Enter key with modern semantics:

  • Dialogs: Allow / Deny choices
  • Installers: Allow installation
  • Permissions: Allow access
  • Commands: Confirm action

Internal mapping: KEY_ALLOW = KEY_ENTER (legacy compatible)

MODE Key

Caps Lock position, system modifier layer:

  • Activates system shortcuts
  • Changes keyboard to system layer
  • KEY_FLAG_MODE modifier

META / вҢҳMOD Keys

  • META: NeolyxOS launcher
  • вҢҳMOD: Command-like modifier for shortcuts

API Usage

// Check if user pressed ALLOW
if (keyboard_is_allow(key)) {
    confirm_action();
}

// Check MODE modifier
uint16_t mods = keyboard_get_modifiers();
if (keyboard_mode_active(mods)) {
    handle_system_shortcut(key);
}

// Convert Shift+Fn to media key
uint8_t media = keyboard_fn_to_media(scancode, shift_held);
if (media) {
    handle_media_key(media);
}

Modifier Flags

Flag Value Description
KEY_FLAG_SHIFT 0x0100 Shift held
KEY_FLAG_CTRL 0x0200 Ctrl held
KEY_FLAG_ALT 0x0400 Alt held
KEY_FLAG_MODE 0x1000 MODE key held
KEY_FLAG_META 0x2000 META key held
KEY_FLAG_MOD 0x4000 MOD key held
