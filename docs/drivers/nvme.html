💾 NVMe Driver Complete

Production-ready NVMe storage driver with I/O queues, PRP lists, and interrupt support.

Files

File Purpose

kernel/src/drivers/storage/nvme.h Structures and API

Features

Controller Initialization

  • PCI device detection (class 0x01, subclass 0x08)
  • BAR0 MMIO register mapping
  • Admin queue setup
  • Controller identify command

I/O Queues

  • Create I/O Completion Queue (nvme_create_io_cq)
  • Create I/O Submission Queue (nvme_create_io_sq)
  • Doorbell ring for submission/completion

PRP List Support (Physical Region Pages)

Transfer Size PRP Method
≤ 4 KB PRP1 only
≤ 8 KB PRP1 + PRP2
> 8 KB (up to 128 KB) PRP1 + PRP list page

Interrupt Support

  • MSI-X capability detection
  • Legacy INTx fallback
  • Interrupt-assisted polling

Per-Core Queue Design (Phase 4)

Designed for multi-core performance with one I/O queue per CPU.

nvme_io_queue_ctx_t *nvme_get_cpu_queue(nvme_controller_t *ctrl);
int nvme_setup_per_core_queues(nvme_controller_t *ctrl, int num_cpus);

Public API

nvme_init()

Scan PCI for NVMe controllers and initialize them.

int nvme_init(void);
// Returns: number of controllers found

nvme_read()

Read blocks from NVMe namespace.

int nvme_read(nvme_controller_t *ctrl, uint32_t nsid, 
              uint64_t lba, uint32_t count, void *buffer);
// Returns: bytes read, or negative error

nvme_write()

Write blocks to NVMe namespace.

int nvme_write(nvme_controller_t *ctrl, uint32_t nsid,
               uint64_t lba, uint32_t count, const void *buffer);
// Returns: bytes written, or negative error

nvme_get_capacity()

Get namespace capacity in bytes.

uint64_t nvme_get_capacity(nvme_controller_t *ctrl, uint32_t nsid);

Usage Example

// Initialize NVMe subsystem
nvme_init();

// Get first controller
nvme_controller_t *ctrl = nvme_get_controller(0);

// Read 8 sectors (4KB) from LBA 0
uint8_t buffer[4096];
int bytes = nvme_read(ctrl, 1, 0, 8, buffer);

// Use block layer abstraction (recommended)
block_device_t *dev = block_find("nvme0n1");
block_read(dev, 0, 8, buffer);

Data Structures

nvme_sqe_t (Submission Queue Entry - 64 bytes)

typedef struct {
    uint8_t  opcode;
    uint8_t  flags;
    uint16_t command_id;
    uint32_t nsid;
    uint64_t reserved;
    uint64_t mptr;
    uint64_t prp1;
    uint64_t prp2;
    uint32_t cdw10, cdw11, cdw12, cdw13, cdw14, cdw15;
} nvme_sqe_t;

nvme_cqe_t (Completion Queue Entry - 16 bytes)

typedef struct {
    uint32_t result;
    uint32_t reserved;
    uint16_t sq_head;
    uint16_t sq_id;
    uint16_t command_id;
    uint16_t status;
} nvme_cqe_t;
