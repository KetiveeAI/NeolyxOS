Calling Convention

Register Convention (x86_64 SYSCALL/SYSRET):
  RAX  → syscall number
  RDI  → arg1
  RSI  → arg2
  RDX  → arg3
  R10  → arg4 (RCX clobbered by SYSCALL)
  R8   → arg5
  RAX  ← return value (negative = -errno)
