Process Control Block (PCB)

typedef struct process {
    uint32_t pid;
    char name[64];
    process_state_t state;      // READY, RUNNING, BLOCKED, ZOMBIE
    
    // CPU context
    cpu_context_t context;      // All registers
    uint64_t kernel_stack;
    uint64_t user_stack;
    
    // Memory
    uint64_t page_table;        // CR3 value
    uint64_t heap_start;
    uint64_t heap_end;
    
    // Scheduling
    uint8_t priority;
    uint64_t time_slice;
    uint64_t cpu_time;
    
    // Links
    struct process *parent;
    struct process *next;
} process_t;

Process States

State Description
PROCESS_READY Waiting to be scheduled
PROCESS_RUNNING Currently executing
PROCESS_BLOCKED Waiting for I/O or event
PROCESS_ZOMBIE Exited, waiting for parent

Context Switch

Implemented in arch.S - saves all registers to old PCB and loads from new PCB.

; context_switch(old, new)
; RDI = old context pointer
; RSI = new context pointer
context_switch:
    ; Save callee-saved registers to old context
    mov [rdi + 0x00], rbx
    mov [rdi + 0x08], rbp
    mov [rdi + 0x10], r12
    ...
    ; Load callee-saved registers from new context
    mov rbx, [rsi + 0x00]
    ...
    ret

Scheduler Algorithm

Round-robin with priority levels:

  • High priority: 1ms time slice
  • Normal priority: 10ms time slice
  • Low priority: 50ms time slice
