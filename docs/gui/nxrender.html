Modules

nxgfx Core

Framebuffer graphics, GPU driver interface, 3D lighting with Blinn-Phong shading, and text rendering with built-in bitmap font.

nxrender

Compositor with surface/layer management, damage tracking, window manager with title bars and dragging, and application framework.

widgets

Button, Label, TextField, VBox, HBox, ScrollView. All widgets use vtable pattern for polymorphism.

layout

Flexbox layout engine with row/column direction, justify, align, gap, and wrap support.

theme

Dark and light themes with color palettes, typography, spacing, and corner radius configuration.

animation

Spring physics, cubic bezier easing, keyframe animations, timelines, and staggered sequences.

Quick Start

// Include single unified header #include <nxrender.h> void on_init(nx_application_t* app, void* data) { // Create window nx_window_t* win = nx_window_create("My App", nx_rect(100, 100, 400, 300), 0); // Create vertical layout nx_vbox_t* vbox = nx_vbox_create(); nx_vbox_set_gap(vbox, 12); // Add widgets nx_label_t* label = nx_label_create("Welcome!"); nx_button_t* btn = nx_button_create("Click Me"); nx_widget_add_child((nx_widget_t*)vbox, (nx_widget_t*)label); nx_widget_add_child((nx_widget_t*)vbox, (nx_widget_t*)btn); // Set as window content nx_window_set_root_widget(win, (nx_widget_t*)vbox); nx_wm_add_window(nx_app_window_manager(app), win); } int main() { nx_app_config_t cfg = nx_app_default_config(); cfg.title = "My App"; cfg.on_init = on_init; nx_application_t* app = nx_app_create(cfg); return nx_app_run(app, framebuffer, pitch); }

3D Lighting

NXRender includes a physically-based lighting system with realistic light behavior:

// Create lighting scene nx_lighting_scene_t* scene = nx_lighting_create(); nx_lighting_set_ambient(scene, nx_vec3(0.1, 0.1, 0.12), 0.1); // Add point light (like a light bulb) nx_light_t bulb = nx_light_point( nx_vec3(0, 2, 0), // Position nx_vec3(1, 0.9, 0.8), // Warm white 2.0 // Intensity ); nx_lighting_add_light(scene, bulb); // Calculate surface color with physics nx_surface_t surface = { .position = pos, .normal = normal, ... }; nx_vec3_t color = nx_lighting_calculate(scene, &surface);

Lighting Features

Feature Description
Inverse Square Law Light dims with distance² (real physics)
Lambert Diffuse Angled surfaces receive less light
Blinn-Phong Specular Shiny highlights that follow viewpoint
Fresnel Effect Surfaces more reflective at grazing angles
Materials Metal, plastic, glass with roughness/IOR

Animation System

High-precision animation with nanosecond timing and CSS-compatible bezier curves:

// Keyframe animation with bezier easing nx_keyframe_anim_t* anim = nx_keyframe_anim_create(500); nx_keyframe_add(anim, 0.0, 0.0, NX_BEZIER_EASE_OUT); nx_keyframe_add(anim, 1.0, 100.0, NX_BEZIER_EASE_IN); nx_keyframe_anim_play(anim); // Spring physics (iOS/Android quality) nx_spring_adv_t* spring = nx_spring_adv_from_preset(NX_SPRING_WOBBLY); nx_spring_adv_set_target(spring, 200); // Each frame while (!nx_spring_adv_is_settled(spring)) { nx_spring_adv_update(spring, dt_ms); widget.x = nx_spring_adv_value(spring); }

GPU Driver Interface

// Initialize GPU driver nx_gpu_driver_t* gpu = nx_gpu_driver_init(); nx_gpu_set_vsync(gpu, true); // Render frame nx_gpu_begin_frame(gpu); nx_gpu_cmd_clear(gpu, NX_COLOR_BLACK); nx_gpu_cmd_rect(gpu, rect, color, 4); nx_gpu_end_frame(gpu); nx_gpu_present(gpu); // VSync wait // Performance uint32_t fps = nx_gpu_get_fps(gpu);

Memory Ownership

NXRender follows strict ownership rules to prevent memory leaks:

  • Parent owns children: When you add a widget as child, parent takes ownership
  • Destroy is recursive: Destroying parent destroys all children
  • Never double-free: Don't destroy children separately after adding to parent
