💿 NXFS - Native Filesystem Complete

NeolyxOS native filesystem with extents, encryption, and checksums.

Files

File Purpose
kernel/src/fs/nxfs/nxfs.c NXFS driver

Features

  • Extent-based allocation - Contiguous block ranges
  • B-tree directories - Fast lookup
  • Optional AES-256 encryption
  • CRC32 checksums - Data integrity
  • Journaling - Crash recovery

Disk Layout

+------------------+
| Superblock       | Block 0
+------------------+
| Block Bitmap     | Blocks 1-N
+------------------+
| Inode Table      | 
+------------------+

+------------------+

Superblock Structure

typedef struct {
    uint32_t magic;           // 0x4E584653 "NXFS"
    uint32_t version;
    uint64_t total_blocks;
    uint64_t free_blocks;
    uint32_t block_size;      // 4096 default
    uint32_t inode_count;
    uint32_t root_inode;
    uint32_t flags;           // ENCRYPTED, JOURNALED
    uint8_t  uuid[16];
    char     label[64];
} nxfs_superblock_t;

Inode Structure

typedef struct {
    uint32_t mode;            // File type + permissions
    uint32_t uid, gid;
    uint64_t size;
    uint64_t blocks;
    uint64_t atime, mtime, ctime;
    uint32_t links;
    uint32_t flags;
    
    // Extent array (inline for small files)
    nxfs_extent_t extents[4];
    uint64_t extent_tree;     // B-tree for large files
    
    uint32_t checksum;
} nxfs_inode_t;

Extent Structure

typedef struct {
    uint64_t logical_block;   // File offset (in blocks)
    uint64_t physical_block;  // Disk location
    uint32_t length;          // Number of blocks
    uint32_t flags;
} nxfs_extent_t;

// Directory operations
