🔒 Security Hardening (Phase 12)

Kernel security, crash prevention, and system integrity.

Components

1. Security Module (security.c)

Capability-based access control with audit logging.

Privilege Levels

Level Value Description
PRIV_USER 0 Normal user processes
PRIV_ADMIN 1 Administrative processes
PRIV_SYSTEM 2 System services
PRIV_KERNEL 3 Kernel mode only

Capabilities (12 flags)

Capability Value Purpose
CAP_SYS_ADMIN 0x01 System administration
CAP_NET_RAW 0x02 Raw network access
CAP_NET_BIND 0x04 Bind privileged ports
CAP_SYS_MOUNT 0x08 Mount filesystems
CAP_SYS_MODULE 0x10 Load kernel modules
CAP_SYS_RAWIO 0x20 Direct I/O access
CAP_SYS_REBOOT 0x40 Reboot system
CAP_FILE_WRITE 0x80 Write system files
CAP_PROC_KILL 0x100 Kill any process
CAP_MEM_LOCK 0x200 Lock memory pages
CAP_DEBUG 0x400 Debug processes
CAP_AUDIT 0x800 Access audit logs

2. Safe Memory (safe_mem.c)

Bounded memory operations to prevent overflows.

// Always use bounded copies
int safe_memcpy(void *dst, uint64_t dst_size, const void *src, uint64_t count);
int safe_strcpy(char *dst, uint64_t dst_size, const char *src);

// Zero-initialized allocation
void *safe_alloc(uint64_t size);

// Poison-on-free (detects use-after-free)
void safe_free(void **ptr_ptr, uint64_t size);

3. Panic Handler (panic_handler.c)

Crash recovery and debugging support.

Panic Types

Type Description
PANIC_ASSERT Assertion failure
PANIC_NULL_PTR Null pointer dereference
PANIC_STACK_OVERFLOW Stack overflow detected
PANIC_DIV_ZERO Division by zero
PANIC_PAGE_FAULT Unhandled page fault

Debug Macros

ASSERT(condition);           // Panic if false
ASSERT_NOT_NULL(ptr);        // Panic if NULL
PANIC("message");            // Immediate panic
SAFE_DIV(a, b);              // Safe division

Usage Example

#include <core/security.h>
#include <core/safe_mem.h>
#include <core/panic_handler.h>

void secure_operation(void *user_ptr, uint64_t size) {
    // Validate pointer
    if (security_validate_pointer(user_ptr, size, 0) != 0) {
        return; // Invalid pointer
    }
    
    // Check capability
    security_context_t *ctx = security_get_context();
    if (security_check_capability(ctx, CAP_FILE_WRITE) != 0) {
        return; // Permission denied
    }
    
    // Safe buffer copy
    char buffer[256];
    safe_memcpy(buffer, sizeof(buffer), user_ptr, size);
    
    // Assert invariants
    ASSERT(buffer[0] != 0);
}
