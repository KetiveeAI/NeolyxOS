# Support

## Getting Help

If you need help with Neolyx OS, here are your options:

### Official Website
Visit [neolyx.ketivee.com](https://neolyx.ketivee.com/) for the latest news, updates, and information.

### Documentation
- [README.md](README.md) - Project overview and features
- [PROJECT_STRUCTURE.md](PROJECT_STRUCTURE.md) - Codebase structure
- [Architecture.md](Architecture.md) - System architecture
- [CONTRIBUTING.md](CONTRIBUTING.md) - How to contribute

### Community Support

#### GitHub Issues
For bug reports and feature requests, please use GitHub Issues:
- [Report a Bug](https://github.com/KetiveeAI/NeolyxOS/issues/new?template=bug_report.md)
- [Request a Feature](https://github.com/KetiveeAI/NeolyxOS/issues/new?template=feature_request.md)

#### Discussions
For questions, ideas, and general discussion:
- Use GitHub Discussions (if enabled)
- Check existing issues for similar questions

### Before Asking for Help

1. Check the documentation
2. Search existing issues
3. Try the latest version
4. Gather relevant information:
   - OS version/commit hash
   - Build environment
   - Error messages and logs
   - Steps to reproduce

### Response Time

This is an open-source project maintained by volunteers. Response times may vary:
- Bug reports: Usually within a few days
- Feature requests: Reviewed periodically
- Pull requests: Reviewed as time permits

### Commercial Support

Currently, no commercial support is available. This project is community-driven.
