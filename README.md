# Neolyx OS

Neolyx OS is a modern, fully functional 64-bit desktop operating system designed from scratch. It features a custom UEFI bootloader, a 64-bit C kernel, a graphical user interface, the NXFS file system, a network manager, a package manager (nxpkg), and support for modern hardware including NVIDIA GPUs and Intel/AMD CPUs.

## Key Features
- 64-bit UEFI bootloader
- Custom C kernel with multitasking and memory management
- NXFS file system
- Full graphical desktop environment
- Network manager and TCP/IP stack
- nxpkg package manager and .app support
- Modular drivers for GPU, input, storage, and network
- Custom icons and file explorer
- Designed for performance, extensibility, and modern hardware
